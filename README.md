                    <p>Of course. You can place FAQs on any page you want by adding a widget in the backend. There are two ways you can do:
                        <br/>- First, You can add a new widget with its type = FAQ questions list and then assign it to the page you want it to display.
                        <br/>- Or you can choose your wanted page and then add a widget with the type = FAQ questions list into its content.</p>
                </div>
                <div>
                    <strong>Does your extension support multiple store views?</strong>
                    <p>Yes, it does.</p>
                </div>
                <div>
                    <strong>Can I limit the number of questions displaying on per page?</strong>
                    <p>Yes. Simply enter your wanted number in the Questions per page field in the backend to limit the number of questions will be shown.</p>
